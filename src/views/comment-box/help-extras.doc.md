### Extras

We have some additional markdown syntax. Right now, we only add spoiler syntax. In the future we'll add some more.

<pre lang="markdown">
```spoiler
Your spoiler content.
Support **markdown** syntax.
## Heading 2
```
</pre>

#### Output

```spoiler
Your spoiler content.
Support **markdown** syntax.
## Heading 2
```

Alternative syntax using `<spoiler>` tag.

```html
<spoiler>
Your spoiler content.
Support **markdown** syntax.
## Heading 2
</spoiler>
```
